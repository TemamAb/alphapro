# Security Policy

## Supported Versions

Only the latest version of the `main` branch is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Alpha-Orion seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do NOT open a public issue.**
2.  Email the details to **ahmiforex@gmail.com**.
3.  Include steps to reproduce the issue and any relevant logs.
4.  The team will acknowledge receipt within 48 hours.

## SLA
We aim to patch critical vulnerabilities within 72 hours of confirmation.
